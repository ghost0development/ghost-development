# Ghost Development

**Software House · Szczecin** — "Nowa Generacja Oprogramowania"

Nowoczesne strony internetowe, aplikacje webowe i desktopowe. Indywidualne podejście, szybkie wdrożenia.

## Tech stack

React · Next.js · TypeScript · Tailwind CSS · Python · PySide6 · Supabase · Three.js

## Development

```bash
npm install
npm run dev
```

Otwórz http://localhost:3000.

## Deploy

Strona deployowana automatycznie na GitHub Pages przy każdym pushu na `master`.

### Wymagane zmienne środowiskowe (GitHub Secrets)

- `NEXT_PUBLIC_SUPABASE_URL` — URL projektu Supabase
- `NEXT_PUBLIC_SUPABASE_ANON_KEY` — anon public key
